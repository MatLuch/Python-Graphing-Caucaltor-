# Graph Visualizer 

A simple graph visualizer built with Python and Pygame. This program allows users to enter coordinate points and see them displayed on a customizable graph.

## Features 

- Creates a coordinate plane with X and Y axes
- Graphs custom X and Y points
- Supports adjustable graph limits
- Converts mathematical coordinates into screen positions
- Real-time rendering using Pygame

## Installation

### Requirements
- Python 3.x
- Pygame
